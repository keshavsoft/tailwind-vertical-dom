tailwind-todo

CLI to scaffold Tailwind starter projects quickly.

Install

npm install -g tailwind-todo

or

npx tailwind-todo

Usage

tailwind-todo

Features

  • Tailwind starter setup
  • Boilerplate generation
  • CLI based workflow
  • Fast project scaffolding
  • Zero dependency
  • Simple folder structure
