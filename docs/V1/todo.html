Complete Execution Flow

How To Use tailwind-todo

Step 1 — Create Express Architecture

First generate the base project structure using the express-todo CLI. 

npx express-todo simple

Step 2 — Go To Public Folder

After generating the project structure, navigate into the Public folder. 

cd Public

Step 3 — Create V1 Folder Manually

Inside the Public folder, create a new folder named V1 manually. 

mkdir V1

Step 4 — Enter V1 Folder

Move into the V1 folder. 

cd V1

Step 5 — Execute tailwind-todo

Now run the Tailwind frontend generator command. 

npx tailwind-todo header

Available Command

CLI Usage 

tailwind-todo v1.3.1

Usage:
npx tailwind-todo <command> [options]

Commands:
header              a simple header

Examples:
npx tailwind-todo header

Tip:
npm i -g tailwind-todo
